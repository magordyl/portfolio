Planner V1 — Story Map

How to use: Drag cards between cells to move them between releases or workflow steps. Click any text (title, release name, notes, skeleton activity, sub-step, card title, card description) to edit inline. Hover a cell for a + button to add a new card. Hover a card for a × to delete. Changes save to your browser automatically.

Copy Share URL copies a URL containing the full current state — open it on any device to load the same map (no backend, no account). Paste the URL back in a Claude conversation and the state can be decoded there too. Export Markdown downloads a clean markdown snapshot. Reset restores the original seeded content. localStorage is per-device — use Share URL to move state between devices.
