Phase 3 · Design Tokens · v2

Design explorer, round 2

Typography, type scale, and card treatment are locked. Palettes have been rethought: instead of "one accent + another accent", each revised palette is a tonal system — one hue expressed as a 12-step scale, with each step assigned a role. This is how Radix Colors and Material 3 work. It solves both your "doesn't feel cohesive" and "not enough colours" concerns in one move.

Locked from round 1

Typography (Option A)

Fraunces × Geist Sans

Characterful flared serif paired with modern narrow sans.

Type scale (Option B, tweaked)

72 / 32 / 16

Hero 72px, case title 32px, body 16px. Big headline, tight case cards, readable body.

Card treatment (Option A)

Soft rounded, hover lifts

8px corners, 1px border, subtle shadow + lift on hover. Interactions refined later.

What I got wrong in round 1, and the fix

Round 1 presented palettes as "primary accent + secondary accent". Two of them clashed (B: royal + sky, D: cobalt + ember) because both colours in each pair were saturated and competing for the "brand" slot. The eye couldn't tell which was primary.

The fix: one hue, twelve steps. Each step has a role — near-surface tint, subtle background, element background, hover state, border, solid brand, hovered solid, low-contrast text, high-contrast text. This is how modern design systems work (Radix, Material 3, Tailwind v4). It gives you twenty usable tokens from one colour, which answers your "is this enough colours for a full portfolio?" question with a yes.

Below: three single-hue options (royal, periwinkle, a refined royal+sand dual) plus one true complementary option (royal + claret) for contrast. Every palette shows a case study card, a build log strip, and a code block (for writing posts) so you can see how the tokens hold up across real content types.

Revised palette A — Royal Tonal

Single hue · 12-step tonal scale

Royal Tonal

Built from royal #3B5BDB (your favourite from round 1). The "periwinkle" you loved in Option E is just step 11 of this scale — #A5B4FF. So your two favourite colours are already the same colour, at different roles: solid brand (step 8) and text link (step 10-11). Surfaces pick up a subtle blue undertone so the page reads cohesive, not "charcoal with a blue floating on top".

Revised palette B — Royal + Sand

Dual hue · warm-biased editorial

Royal + Sand

Royal #3B5BDB for interactive (links, CTAs, brand). Warm sand #D4B896 for editorial flourish — italic dates, kicker labels, "Selected work" numerals. This is the replacement for Option D (cobalt + ember). Sand is desaturated (not saturated amber), so it sits comfortably beside royal instead of fighting it. Uses the original warm charcoal base you liked — most editorial-feeling of the options.

Revised palette C — Periwinkle Tonal

Single hue · softer anchor

Periwinkle Tonal

Same system as palette A, but anchored on periwinkle #8B9CFF as the solid brand step instead of royal. Softer, more restrained, leans literary. The deep royal you loved still lives in the scale at step 8 #4A4A82 as the "active / pressed" state. Good if you want the overall tone to read as *quiet confidence* rather than *bold product*.

Revised palette D — Royal + Claret

Dual hue · true complementary

Royal + Claret

Royal #3B5BDB for interactive. Deep claret / wine #8E3A4A for editorial flourish — kicker labels, italic metadata, "NEW" tags. Royal and claret are roughly on opposite ends of the colour wheel, so they don't compete for the same brand slot (unlike royal + sky). More dramatic than sand; feels like a literary quarterly. Risk: could read as sports-team-ish if over-used.
